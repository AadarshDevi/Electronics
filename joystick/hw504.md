# HW504 Joystick Module

## Components
1. ESP32-WROVER-E
2. HW504 Joystick Module
3. Jumper Wires

## Connection(s)
| HW504 | ESP32 | Wire Color |
|:-------:|:--------:|:-------:|
| VRx (X Axis) | GPIO 14 (ADC2 Pins) | Orange |
| VRy (Y Axis) | GPIO 12 (ADC2 Pins) | Yellow |
| SW (Button Press) | GPIO 13 (ADC2 Pins) | Dusty Brown Water |
| VCC | 3V3 | Green |
| GND | GND | White |

## Programming
1. Create Pins
```c
const int xAxisPin = 14;
const int yAxisPin = 12;
const int buttonPin = 13;
```

2. Joystick Button uses internal pullup resistor & is an input
```c
pinMode(xybPins[2], INPUT_PULLUP);
```

3. read values
```c
xVal = analogRead(xybPins[0]);
yVal = analogRead(xybPins[1]);
buttonVal = digitalRead(xybPins[2]);
```

4. Convert to voltage. ESP32 runs on 3V3
```c
float xVolt = xVal / 4095.0 * 3.3;
float yVolt = yVal / 4095.0 * 3.3;
float buttonVolt = buttonVal * 3.3;
```
