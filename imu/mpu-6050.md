# MPU-6050 IMU

## Components
1. ESP32-WROOM-32UE
2. MPU-6050
3. Female to Female Jumper Wires

## Connection(s)
| MPU | ESP32  |  Wire  |
|:---:|:------:|:------:|
| VCC | 3.3V   | Red    |
| GND | GND    | Black  |
| SDA | GPIO21 | Orange |
| SCL | GPIO22 | Yellow |

## Library Manager
Install MPU605 and i2cdevlib libs by _**Jeff Rowberg**_
1. Goto this [GitHub Repo](https://github.com/jrowberg/i2cdevlib)
2. Click Code > Download ZIP
3. Extract teh zip
4. Go in to Arduino
5. Copy Folders: MPU6050 & I2Cdev
6. Find your Arduino IDE Libraries folder
7. Paste the folders
8. Restart Arduino IDE
