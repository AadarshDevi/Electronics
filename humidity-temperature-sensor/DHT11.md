# DHT11 Humidity & Temperature Sensor

## Connection

| DHT11 Pin |       ESP32 Pin      | Wire Color |
|:---------:|:--------------------:|:----------:|
|    GND    |         GND          |    Black   |
|    Data   |     Digital Pin 21   |    Green   |
|    VCC    |         5V           |    Red     |
